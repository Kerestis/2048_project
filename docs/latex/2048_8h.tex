\doxysubsection{Подробное описание}
Заголовочный файл для консольной игры 2048. 

Содержит определения глобальных переменных и функций, отвечающих за основную логику игры\+: загрузка/сохранение состояния, отрисовка, управление игровым полем, генерация новых чисел, обработка ходов и проверка завершения игры.
